# Сборка
Пока что достаточно
```
g++ main.cpp -o main
```

# Использование
В этом репозитории код для нескольких задач. Название задачи необходимо написать первым аргументом командной строки (см. раздел нужной задачи). Далее можно перечислить необязательные параметры:
* `-d <detail_level>` - Уровень подробности вывода во время работы. Целое число от 0 до 3.
* `-i <max_iterations>` - Максимальное количество шагов локального поиска. Целое число.
* `-n <number_of_launches>` - Число независимых запусков поиска, среди результатов которых выбирается наилучший. Целое число.

# 1. Задача о рюкзаке методом локального поиска
* Запустить `./main knapsack`.
* Ввести в стандартный поток ввода входные данные.

Формат вывода:
* "Best point:"
* {Битовая строка -- взятые предметы}
* `score:` {стоимость результата} `weight:` {Вес получившегося рюкзака}

# 2. Задача о коммивояжёре методом локального поиска
* Запустить `./main tsp`.
* Ввести в стандартный поток ввода входные данные.

Формат вывода:
* "Best point:"
* {Числа через пробел -- индексы вершин во введённом массиве. Расположены в порядке, в котором их нужно обойти}
* `length:` {длина пути}
